URL Shortener API Documentation

Create Short URL

Endpoint: POST /api/shorturls/

Request Body:

{
    "original_url": "https://example.com/very/long/url",
    "custom_code": "optional-custom-code"
}

Response:

{
    "id": 1,
    "original_url": "https://example.com/very/long/url",
    "short_code": "abc123",
    "short_url": "https://yoursite.com/abc123",
    "click_count": 0,
    "created_at": "2023-01-01T12:00:00Z"
}

Get Short URL Info

Endpoint: GET /api/shorturls/{short_code}/

Response:

{
    "id": 1,
    "original_url": "https://example.com/very/long/url",
    "short_code": "abc123",
    "short_url": "https://yoursite.com/abc123",
    "click_count": 5,
    "created_at": "2023-01-01T12:00:00Z"
}

Get Redirect Info

Endpoint: GET /api/redirect/{short_code}/

Response:

{
    "original_url": "https://example.com/very/long/url",
    "short_code": "abc123",
    "click_count": 5
}
